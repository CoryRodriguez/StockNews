  **2. Helper functions:**

  - `startOfDayET(dateET: string): Date` — converts "YYYY-MM-DD" to UTC Date at midnight ET. Use `new Date(\`${dateET}T05:00:00Z\`)` for EST (conservative; DST shift is ±1hr which still captures full trading day when combined with endOfDayET).
  - `endOfDayET(dateET: string): Date` — startOfDayET + 24 hours.
  - `computeScore(data: { totalPnl, winRate, firedCount, signalCount, dailyLossLimitHit, adherencePct }): number` — weighted composite 0-100 per RESEARCH.md formula:
    - P&L vs baseline (30%): `clamp((totalPnl / 200) * 30, 0, 30)`
    - Win rate (25%): `winRate * 25` (winRate is 0-1)
    - Signal quality (20%): `(firedCount / max(signalCount, 1)) * 20`
    - Risk compliance (15%): 15 if no daily-loss-limit breaches, else 0
    - Strategy adherence (10%): `adherencePct * 10`
    - All clamped to max. Round to integer.
  - `generateSuggestions(data: DailyRecapData): string[]` — rule-based coach-like suggestions (max 3):
    1. Missed opportunity pattern: if 2+ rejected signals moved +5%, group by reason, surface top pattern
    2. Catalyst strength: if any category has 2+ trades with 100% win rate, note it
    3. Catalyst weakness: if any category has 2+ trades with 0% win rate, note it
    4. Early exit pattern: if >50% of trades exited via hard_stop, suggest reviewing stop level
    5. Overhold pattern: if >50% of trades exceeded recommended hold by >50%, note it

  **3. Core `computeRecap(dateET: string): Promise<DailyRecapData>` function:**

  Uses `Promise.all` to query in parallel:
  - `prisma.botTrade.findMany({ where: { status: 'closed', exitAt: { gte: startOfDayET(dateET), lt: endOfDayET(dateET) } }, orderBy: { exitAt: 'asc' } })`
  - `prisma.botSignalLog.findMany({ where: { evaluatedAt: { gte: startOfDayET(dateET), lt: endOfDayET(dateET) } } })`
  - `prisma.botDailyStats.findFirst({ where: { date: dateET } })`
  - `prisma.strategyRule.findMany()`
  - `getSnapshots(['SPY', 'QQQ']).catch(() => [])` — benchmark data via changePct field

  IMPORTANT: Filter closed trades by `exitAt` (not `createdAt`) per Pitfall 1 from RESEARCH.md.

  Assembles:
  - **Summary:** P&L, win/loss, score, signal counts, best/worst trade, benchmarks
  - **Trades section:** Map each BotTrade to TradeBreakdownRow using entryVwapDev, peakPrice, maxDrawdownPct fields
  - **Signals section:** Build rejectionHistogram from BotSignalLog rejectReason counts; missedOpportunities = signals where postRejectPeakPct >= 5
  - **Catalysts section:** Group trades by catalystType, compute per-category P&L and win rate
  - **Adherence section:** For each trade, look up StrategyRule by catalystCategory. Compare actualHoldSec (exitAt - entryAt in seconds) vs recommendedHoldSec. Within ±20% = "on-target", <80% = "early-exit", >120% = "overhold". Build exitReasonDistribution from all trade exitReasons.
  - **Suggestions:** Call generateSuggestions()
  - **Benchmarks:** SPY/QQQ changePct from snapshots, plus rolling self-averages from prior DailyRecap rows

  **4. `persistRecap(dateET: string): Promise<void>` function:**

  Calls `computeRecap(dateET)`, then `prisma.dailyRecap.upsert({ where: { date: dateET }, update: { ...scalars, sectionsJson: sections as Prisma.InputJsonValue, computedAt: new Date() }, create: { date: dateET, ...scalars, sectionsJson: sections as Prisma.InputJsonValue } })`.

  **5. Self-average computation (for benchmarks):**

  ```typescript
  async function getSelfAverages(dateET: string, days: number): Promise<{ pnl: number; winRate: number; score: number; signalCount: number } | null> {
    const rows = await prisma.dailyRecap.findMany({
      where: { date: { lt: dateET } },
      orderBy: { date: 'desc' },
      take: days,
      select: { totalPnl: true, winRate: true, score: true, signalCount: true },
    });
    if (rows.length === 0) return null;
    return {
      pnl: rows.reduce((s, r) => s + r.totalPnl, 0) / rows.length,
      winRate: rows.reduce((s, r) => s + r.winRate, 0) / rows.length,
      score: rows.reduce((s, r) => s + r.score, 0) / rows.length,
      signalCount: rows.reduce((s, r) => s + r.signalCount, 0) / rows.length,
    };
  }
  ```

  ```typescript
  // GET /api/bot/recap?date=YYYY-MM-DD — returns full recap for a single day
  router.get('/recap', requireAuth, async (req, res) => {
    try {
      const dateET = (req.query.date as string) ?? getTodayDateET();

      // Try persisted first (fast)
      const persisted = await prisma.dailyRecap.findUnique({ where: { date: dateET } });
      if (persisted) { res.json(persisted); return; }

      // Fall back to on-demand computation (slower)
      const data = await computeRecap(dateET);
      res.json(data);
    } catch (err) {
      console.error('[BotRoute] /recap error:', err);
      res.status(500).json({ error: 'Failed to load recap' });
    }
  });

  // GET /api/bot/recap/history?mode=week|month&anchor=YYYY-MM-DD
  router.get('/recap/history', requireAuth, async (req, res) => {
  ```

  ```typescript
  function getWeekDates(anchor: string): string[] {
    const d = new Date(`${anchor}T12:00:00Z`);
    const day = d.getUTCDay(); // 0=Sun
    const monday = new Date(d);
    monday.setUTCDate(d.getUTCDate() - ((day + 6) % 7)); // go back to Monday
    const dates: string[] = [];
    for (let i = 0; i < 5; i++) {
      const cur = new Date(monday);
      cur.setUTCDate(monday.getUTCDate() + i);
      dates.push(cur.toISOString().slice(0, 10));
    }
    return dates;
  }

  function getMonthDates(anchor: string): string[] {
    const [year, month] = anchor.split('-').map(Number);
    const dates: string[] = [];
    const daysInMonth = new Date(year, month, 0).getDate();
    for (let d = 1; d <= daysInMonth; d++) {
      const dateStr = `${year}-${String(month).padStart(2, '0')}-${String(d).padStart(2, '0')}`;
      // Only include weekdays (Mon-Fri)
      const dayOfWeek = new Date(`${dateStr}T12:00:00Z`).getUTCDay();
      if (dayOfWeek >= 1 && dayOfWeek <= 5) dates.push(dateStr);
    }
    return dates;
  }
  ```
